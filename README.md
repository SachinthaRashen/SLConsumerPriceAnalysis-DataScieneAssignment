# Sri Lankan Supermarket Price Analysis

## Project Overview
This project analyzes the pricing of essential consumer goods across major Sri Lankan supermarket chains (Keells, Cargills, Arpico, etc.). It uses Machine Learning to predict the total cost of a grocery basket based on individual item prices.

## Tech Stack
* **Language:** Python
* **Environment:** Google Colab / Jupyter Notebook
* **Libraries:** Pandas, Scikit-Learn, TensorFlow (Keras)

## Key Findings
* **Best Model:** Linear Regression (R²: 0.7151)
* **Insights:** While Neural Networks are powerful, simple linear relationships in pricing data are often best captured by traditional regression models.

## How to Run
1. Clone this repository.
2. Ensure `sl_supermarket_data.csv` is in the same directory as the notebook.
3. Install dependencies using `pip install -r requirements.txt`.
4. Open `Assignment_Notebook.ipynb` in Jupyter or Google Colab.
